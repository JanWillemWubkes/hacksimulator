Leer Ethisch Hacken in de Browser

HackSimulator.nl is een gratis, Nederlandstalige terminal simulator waarmee je veilig leert hoe cybersecurity professionals werken. Alle commands draaien lokaal in je browser — er worden geen verbindingen gemaakt met externe servers. Je kunt onmogelijk per ongeluk echte systemen aanvallen. Perfect voor studenten, career switchers en iedereen die nieuwsgierig is naar ethisch hacken.

Populaire Commands

nmap Netwerk

Scan netwerken en ontdek open poorten. De meest gebruikte tool voor network reconnaissance.

ls Basis

Bekijk bestanden en mappen in de huidige directory. Het startpunt van elke terminal sessie.

hashcat Security

Analyseer wachtwoord hashes en leer waarom sterke wachtwoorden essentieel zijn.

grep Basis

Doorzoek bestanden op specifieke tekst. Onmisbaar voor log analyse en forensisch onderzoek.

sqlmap Security

Test websites op SQL injection kwetsbaarheden. Leer hoe databases aangevallen en beschermd worden.

wireshark Netwerk

Analyseer netwerkverkeer en begrijp hoe data over het internet reist.

Zo Begin Je

1
Type help

Krijg een overzicht van alle beschikbare commands met korte beschrijvingen in het Nederlands.

2
Verken het filesystem met ls en cd

Navigeer door de virtuele mappenstructuur en ontdek verborgen bestanden en directories.

3
Probeer security tools zoals nmap

Scan virtuele netwerken, analyseer poorten en leer denken als een ethical hacker.

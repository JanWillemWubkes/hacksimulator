Expected Impact Analysis

Scannability +25% - Sans-serif H1/H2 veel sneller te scannen in blog overview
Readability +15% - Title case + reduced letter-spacing = natuurlijker leesritme
Terminal Aesthetic 95% → 65% - Blog-appropriate balance, H3/code/colors blijven
Visual Hierarchy +40% - Size + weight + family = duidelijke heading differentiation
Info Box Visibility +50% - Opacity 0.05→0.1 + border 3px→4px = beter onderscheid
Link Refinement Modern look - underline-offset creates breathing room
Professional Score 7/10 → 9/10 - Industry-aligned zonder merk DNA te verliezen
Card Shadows Subtle depth - LogRocket pattern (4px 12px blur) vs current glow

Key Changes Breakdown

Wave 1: Typography Revolution

  • H1/H2: Courier New → System sans-serif (scannability)
  • H3: KEEPS Courier New (terminal accent marker)
  • Sizes: Arbitrary → Modular 1.25 ratio (39/31/25/20/16px)
  • Case: UPPERCASE → Title Case / Sentence case
  • Spacing: 2px/1px letter-spacing → 0.5px/0px
  • Weight: H1/H2 bold (700), H3 semi-bold (600)

Wave 2: Content Organization (not in mockup)

  • Categories: 4 filters (Beginners/Gevorderden/Tools/Concepten)
  • Progress Bar: CSS-only scroll indicator (8min+ posts)
  • Related Posts: 2-3 suggestions per post footer
  • Metadata: Better visual hierarchy (author/date/time)

Wave 3: Visual Refinement

  • Shadows: 0 4px 20px rgba(0,255,136,0.1) → 0 4px 12px rgba(0,0,0,0.04)
  • Links: Added underline-offset 3px + thickness 1px
  • Info Boxes: Opacity 0.05 → 0.1, border 3px → 4px
  • Professional Polish: LogRocket/CSS-Tricks patterns

What STAYS (Terminal DNA)

  • Colors: Cyan links, green success, orange warnings
  • ASCII Brackets: [ TIP ], [ ! ], [ ✓ ]
  • Code Blocks: Monospace, terminal styling, prompt/output
  • H3 Headings: Monospace (technical topic marker)
  • Dark Frame: Navbar/footer always dark
  • Body Text: 1.1rem, 1.7 line-height (already perfect!)

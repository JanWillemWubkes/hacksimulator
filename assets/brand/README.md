# HackSimulator.nl — Brand Assets

Het logo is een **H-monogram** (Hack) opgebouwd uit terminal-primitieven: de letter H
staat op een `_` command-line-balk. Eén beeld, twee betekenissen — de naam én de terminal.

## Bestanden

| Bestand | Gebruik |
|---|---|
| `logo.svg` | **Primair.** Groene tegel + donkere H. Op lichte/neutrale achtergrond, als app-icoon of avatar. Vector — schaalt oneindig. |
| `logo-on-dark.svg` | Groen glyph zónder tegel, transparant. Voor **donkere** achtergronden (zoals de site-navbar). |
| `logo-mono-black.svg` | Eenkleurig zwart glyph, transparant. Print, facturen, eenkleurig licht gebruik. |
| `logo-mono-white.svg` | Eenkleurig wit glyph, transparant. Over foto's of donker-eenkleurig gebruik. |
| `logo-256.png` `logo-512.png` `logo-1024.png` | Raster-exports van de primaire tegel. Voor plekken die geen SVG accepteren (sommige social-profielen, presentaties). |

> **Tip:** gebruik bij voorkeur de **SVG's** — die blijven scherp op elk formaat. Pak een PNG
> alleen als een platform geen SVG accepteert.

## Merkkleuren

| Naam | Hex | Gebruik |
|---|---|---|
| Neon-groen | `#9fef00` | Accent, glyph, `.nl` in de wordmark |
| Donker | `#0d1117` | Achtergrond, glyph-op-tegel |
| Zacht-wit | `#c9d1d9` | Wordmark-tekst op donker |

## Wordmark

De volledige merknaam is **HackSimulator.nl**, gezet in **Space Grotesk** (semibold).
Het `.nl`-deel staat in neon-groen (`#9fef00`), de rest in zacht-wit op donker / donker op licht.

## Niet doen

- Het glyph uitrekken of de verhoudingen wijzigen.
- Andere kleuren dan de merkkleuren gebruiken voor het glyph.
- De groene tegel op een groene achtergrond plaatsen (gebruik dan `logo-on-dark.svg`).
